# Simple Future

it + a + infinitivo

| person              | ir    | a   | Ifinitivo |
| ------------------- | ----- | --- | --------- |
| yo                  | voy   |     | apprender |
| tú                  | vas   |     | dormir    |
| él/ella/Usted       | va    | a   | descansar |
| nosotros/-as        | vamos |     |           |
| vosotros/-as        | vais  |     |           |
| ellos/ellas/Ustedes | van   |     |           |

- voy a dormir
