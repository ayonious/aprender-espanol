# Present Continuous

It means Im doing this right now.

| person              | estar   | Verb(-ar) | Verb(-ir) | Verb(-er)   |
| ------------------- | ------- | --------- | --------- | ----------- |
| yo                  | estoy   | hablando  | muriendo  | aprendiendo |
| tú                  | estás   | empezando | durmiendo |             |
| él/ella/Usted       | está    |           |           |             |
| nosotros/-as        | estamos |           |           |             |
| vosotros/-as        | estáis  |           |           |             |
| ellos/ellas/Ustedes | están   |           |           |             |
